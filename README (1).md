# 9700/42 — Microprocessors & Interfacing Study Site

A pastel-themed visual study guide for the Microprocessors & Interfacing course.

## Files
- `index.html` — Main page
- `style.css` — All styles
- `script.js` — Scroll animations

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `microprocessors-notes`)
2. Upload all 3 files
3. Go to **Settings → Pages → Source → main → Save**
4. Live at: `https://YOUR-USERNAME.github.io/microprocessors-notes/`

No build tools needed!
